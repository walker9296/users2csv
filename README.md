# users2csv
discord的bot程序，以csv格式输出服务器/频道用户的一览表

# 前提条件
- python 3.6+
- 安装discord.py
```
pip install -U discord.py
```
- 在 [discord](https://discord.com/developers/applications/) 创建bot
- 选择discord服务器给bot授权

# 启动bot后台程序
```
#python userlistbot.py [bot's TOKEN]
```
